RTScheduler
 Aim : To generate reports reqularly as per the schedule using the report templates(*.prpt files) generated by Pentaho Report Designer 3.9 and then mail the generated reports to the concerned persons.
 
 Build : To build this project use the reqular command we use to build other projects. When the build is successfull, RTScheduler.war is created in target folder of RTScheduler Project.
 
 Deployment Steps :
	1) Copy the RTScheduler.war to the tomcat webapps folder. 
	2) Following properties in /apache-tomcat-6.0.33/webapps/RTScheduler/WEB-INF/classes/rtscheduler.properties should be changed accordingly
	   Property                       				Property Description              																							Example
	   dailyReportsInputDir			  				Path of the input directory where report templates exisit.																	dailyReportsInputDir=C:\\apache-tomcat-6.0.33\\webapps\\RTScheduler\\WEB-INF\\smartrt
	   dailyReportsOutputDir          				Path of the Output directory where generated reports should be stored.    													dailyReportsOutputDir=C:\\Users\\Pradeep\\Desktop\\dailyReportsDir
	   mfino.mail.server              				Mail Server Name 																											mfino.mail.server=smtp.gmail.com
	   mail.server.auth_name          				Sender Email Id 																											mfino.mail.server.auth_name=mfinoemailtest@gmail.com
	   mfino.mail.server.auth_password 				Sender Email Password 																										mfino.mail.server.auth_password=User1234
	   mfino.mail.server.from_name     				Sender Name 																												mfino.mail.server.from_name=dev.mfino.com
	   mfino.mail.server.port        				Mail Server Port 																											mfino.mail.server.port=465
	   mfino.mail.server.require_auth 				Does the Mail Server Requires Authentication?  																				mfino.mail.server.require_auth=true
	   mfino.mail.server.require_ssl   				Does the Mail Server Requires ssl?																							mfino.mail.server.require_ssl=true
	   email.recipient.list           				Concerned people emailIds seperated by ',' who needs the reports to be sent to their Email ID after report generation. 		email.recipient.list=pradeep@mfino.com,mfinoemailtest@gmail.com
	   
	   
	   
	3) To change the schedule of RTScheduler, change the following line in /apache-tomcat-6.0.33/webapps/RTScheduler/WEB-INF/applicationContext.xml as per your requirements
		<property name="cronExpression" value="0 * 2 * * ?" />
		
    4) Add the following resource tag for DB configuration in the following location:
       ..\apache-tomcat-6.0.33\conf\context.xml                                                                                                                             
                                     
	    <Resource defaultTransactionIsolation="-1" defaultAutoCommit="false"
	    maxActive="15" maxIdle="3" initialSize="1"
	    username="root" password="mFino260"
	    url="jdbc:mysql://mfinodbserver:3306/mfino"
	    driverClassName="org.gjt.mm.mysql.Driver"
	    type="javax.sql.DataSource" auth="Container"
	    name="jdbc/mysql_db"/>
	    
	    and modify the properties accordingly.
